# Ninja Sites 🏯

Smart websites with ctrl-a pre-wired AI integration.

## What Is This?

Ninja Sites is a modern web framework built on Next.js that comes with ctrl-a (AI teammate) built-in. Every site gets its own AI that knows the business, can chat with visitors, handle tasks, and more.

## Features

- **ctrl-a Integration** — Pre-wired AI teammate for every site
- **Modern Stack** — Next.js, TypeScript, Tailwind CSS
- **Modular** — Dashboard, Chat, CRM, Analytics, Projects, Tasks, Settings
- **Deploy Ready** — Vercel, Netlify, anywhere Next.js runs

## Products Built on Ninja Sites

- **TOR 90 Tracker** — Squad accountability for the Career Accelerator 90 program

## Getting Started

```bash
cd ninja-sites
npm install
npm run dev
```

## Architecture

```
src/
├── app/           # Next.js App Router pages
├── components/   # Reusable UI components
├── hooks/        # Custom React hooks
└── lib/          # Utilities and helpers
```

## ctrl-a Integration

Every Ninja Site connects to ctrl-a Workspace API for AI capabilities. Configure in .env.local.
